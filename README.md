# 📄 AI ATS Resume Analyzer

## 📌 Project Overview

AI ATS Resume Analyzer is an AI-powered resume evaluation system that analyzes resumes against job descriptions and calculates ATS compatibility scores using Natural Language Processing techniques.

---

## 🚀 Problem Statement

Many candidates fail ATS screening because:

* Missing keywords
* Improper resume formatting
* Weak skill matching
* Poor resume optimization

This system helps candidates optimize resumes for ATS systems.
